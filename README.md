# Team Portfolio

A collaborative full-stack portfolio website built in 6 days by **Anmol** (Frontend) and **Priyansu** (Backend).

## 📋 Project Overview

This project showcases a complete team portfolio with:
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB
- **Deployment**: Vercel (Frontend) & Render (Backend)

## 🚀 Quick Start

### Prerequisites
- Node.js v16 or higher
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The development server will run on `http://localhost:3000`
